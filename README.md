# Revisa-
O projeto consiste no desenvolvimento de um sistema mobile de planejamento de estudos baseado na  técnica de revisão espaçada, com o objetivo de auxiliar estudantes na retenção de conteúdo ao longo do  tempo. A aplicação permitirá o cadastro de matérias e conteúdos de estudo, organizando automaticamente  revisões de intervalos estratégicos.
